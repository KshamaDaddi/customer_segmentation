## 📌 Overview

An end-to-end machine learning system that:
- **Segments** e-commerce customers into 4 behavioural groups using **K-Means clustering**
- **Generates AI-powered business insights** for each segment using **Groq AI (free)**
- **Visualises** live predictions on an interactive **Streamlit dashboard**
- **Exports data** to Power BI for advanced reporting

## 🚀 Setup & Run

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/customer-segmentation.git
cd customer-segmentation
```

### 2. Create a virtual environment

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Mac / Linux
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Get your FREE Groq API key

1. Go to **[console.groq.com](https://console.groq.com)**
2. Sign up with Google or GitHub *(no credit card needed)*
3. Click **API Keys → Create API Key**
4. Copy the key (starts with `gsk_...`)

### 5. Configure environment variables

```bash
cp .env.example .env
```

Open `.env` and paste your key:
```
GROQ_API_KEY=gsk_your_actual_key_here
```

### 6. Add your trained models

Place your trained model files in the `models/` folder:
```
models/
├── kmeans.pkl
└── scaler.pkl
```

> These are generated by your training script using `pickle.dump(...)` or `joblib.dump(...)`

### 7. Run the application

**Terminal 1 — Start the backend:**
```bash
python backend.py
```
Backend will start at `http://127.0.0.1:8000`  
API docs available at `http://127.0.0.1:8000/docs`

**Terminal 2 — Start the frontend:**
```bash
streamlit run frontend.py
```
Dashboard will open at `http://localhost:8501`

## 🔌 API Endpoints

| Method | Endpoint        | Description                              |
|--------|-----------------|------------------------------------------|
| POST   | `/predict`      | Predict customer segment                 |
| POST   | `/ai-insights`  | Get AI analysis for a segment            |
| POST   | `/ai-chat`      | Ask a free-form business question        |
| GET    | `/data`         | Get all recorded predictions (JSON)      |
| GET    | `/powerbi`      | Get all data as CSV (for Power BI)       |
| GET    | `/docs`         | Interactive Swagger API documentation    |

### Example: Predict a customer

```bash
curl -X POST "http://127.0.0.1:8000/predict?age=28&income=75000&spending_score=65"
```

Response:
```json
{
  "segment_name": "High Value",
  "cluster_id": 2
}
```

## 📊 Connect Power BI

1. Open **Power BI Desktop**
2. Click **Home → Get Data → Web**
3. Enter URL: `http://127.0.0.1:8000/powerbi`
4. Click **OK → Load**
5. Build charts using columns: `age`, `income`, `spending_score`, `segment`, `timestamp`
6. Refresh live data: **Home → Refresh**

> ⚠️ Keep `backend.py` running whenever you use Power BI.

## 📋 Segment Analysis (ML)

- **Algorithm:** K-Means Clustering (k=4)
- **Features:** Age, Annual Income, Spending Score
- **Preprocessing:** StandardScaler normalisation
- **Evaluation:** Silhouette Score
- **Model persistence:** joblib (.pkl files)
